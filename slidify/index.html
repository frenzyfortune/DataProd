Iris Random Forest model

Motivation

This app was built to enable the prediction of flower species from a set of four measured features. It uses a random forest model built on the iris dataset.

Building and saving the model

library(randomForest)
library(caret)
data(iris)
inTrain <- createDataPartition(iris$Species,p=0.7,list=F)
train <- iris[inTrain,]
test <- iris[-inTrain,]
fit <- train(Species~.,data=train,method="rf")
save(fit,file="rfmodel.RData")

Model accuracy

First, we may check how good this model performs on the test dataset.

table(test$Species,predict(fit,newdata=test))

##             
##              setosa versicolor virginica
##   setosa         15          0         0
##   versicolor      0         14         1
##   virginica       0          0        15

Not bad!
